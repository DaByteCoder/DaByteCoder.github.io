How to Embedded a Video onto Your Website

Step #1: Choose Your Video

Find the video you want on Youtube (not all will work).

Step #2: Get the Embedded Code

Click on your video, and once you are on the watching screen, go under the video and click Share and then Embed. A screen should pop up with the code, at which point you can click the copy button underneath.

Step #3: Put it on Your Website

Go to your coding space, and paste the code in the body wherever you want it, but make sure you put it in a div of class container. Also, change the width and height to 100% IF you want to change the size later on. If not, keep it how it is.

Step #4: Change the size if wanted

Make a css file to link to your website, and in it declare the wanted height and width for .container using a 16:9 Ratio Calculator.
